# flaskProject
### Flask 中使用 Echarts 3 组件，生成客户账单柱状图
运行环境：Python 2.7.x

#### 使用方法
```bash
$ pip install -r requirements-dev.txt
$ python manage.py runserver -h=0.0.0.0 -p=5000
# 默认 127.0.0.1:5000
```
